Settings > Withdrawal Settings

Check this to automatically disburse payments to vendors on request, no admin approval required. Auto disbursement only works or auto-payment gateways e.g PayPal, Stripe etc. Bank Transfer or other non autopay mode always requires approval, as these are manual transactions.

Allowed order statuses for which vendor may request withdrawal.

Minimum balance to make a withdrawal request. Leave blank to set no minimum limits.

Withdraw threshold days, (make order matured to make a withdraw request.) Leave blank to inactivate this option.

Reverse Withdrawal Settings

Enable this to keep track reverse withdrawals. Incase vendor receive full payment (eg COD) from customer then they have to reverse-pay admin commission. This is only applicable for reverse withdrawal payment method.
